This Plugin creates a nice looking and mobile friendly sidemenu. It requires the Divi Theme.

== Description ==

### Your visitors will love it
Make it easy to navigate your webpage on the go. if the device is small enough, Offcanvas menu let's the visitor click on a menu button and shows a beautiful side-menu using dropdown submenu for navigation. There will be an arrow on the right to enfold a submenu. Configure the device size to control if the menu appears on certain devices.

### It works automatically
Offcanvas Menu will convert any existing menu to a good looking and mobile friendly navigation.
Restyle it with the WordPress theme customizer and make it adopt to your website's style.

### Easy to customizer and tweak it
Use the methods you already know, just open the theme customizer and click _Header and Navigation / OffCanvas-Menu_ to change the look and feel of your sidemenu.

### Requirements for this Plugin
We support up to three layers of nested sub-menu, please stay under the recommended limit.
You will need to use the divi-theme in order to use this plugin.

== Installation ==

1. Log in to /wp-admin.
1. Visit Plugins -> Add New
1. Search for 'AWEOS Offcanvas Menu'
1. Download the plugin.
1. Activate the plugin.
1. Look at your new navigation
